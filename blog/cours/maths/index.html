Définitions

  • Espace des réalisations $\Omega$ : ensemble des valeurs que peut donner un phénomène aléatoire. Il peut être fini ou dénombrable (discret) ou non-dénombrable (continu)
  • Evènement : sous-ensemble de l'espace des réalisations. Il est élémentaire si c'est un sigleton. $\not 0$ est impossible, $\Omega$ est certain.
  • Exemple : $D_8 = {1,2,3,4,5,6,7,8}$, $A = {2,4,6,8}$ (définit par extension), $A$ : $D_8$ pair (définit par compréhension), $X$ v.a. continue, $A$ : $X\in[1,2]$ (définit par extension)
  • Probabilité : aussocie à tout sous-ensemble (ou certains sous-ensembles) de $\Omega$ une valeur entre 0 et 1.
  • Classique (18e siècle) : nombre de cas favorables sur nombre total de cas : $\mathbb{P}(A) = \frac{\text{Card}(A)}{\text{Card}(N)}$
  • Nécessite l'équiprobabilité
  • Nécessite $\Omega$ fini
  • Fréquentiste : fréquence limite de l'évènement : $\mathbb{P}(A) = \lim\limits_{N\rightarrow\infty} \frac{N_A}{N}$
  • Axiomatique :
  • (i) $\mathbb{P}(A)\geq 0$
  • (ii) $\mathbb{P}(\Omega) = 1$
  • (iii) $A\cap B = \not 0 \Rightarrow \mathbb{P}(A\cap B) = \mathbb{P}(A) + \mathbb{P}(B)$

Espace probabilisé (1.5.3)

Un espace probabilisé ou espace de probabilité est un triplet ${\Omega, \mathcal{F}, \mathbb{P}}$ * $\Omega$ est l'espace des réalisations * $\mathcal{F}$ est un sous-ensemble clos pour le complémentaire, ainsi que l'intersection et l'union dénombrables de parties de $\Omega$ : ce peut être l'ensemble de ses parties. C'est une tribu sur $\Omega$; il contient $\not 0$ et $\Omega$. * $\mathbb{P}$ est une application (une mesure) de $\mathcal{F}$ dans $[0,1]$ qui vérifie : * $\mathcal{P}(\Omega) = 1$ * Si les $A_i$ sont deux à deux disjoints : $$\mathbb{P}\left( \bigcup_i A_i \right) = \sum_i \mathbb{P}(A_i)$$

Formules

  • $\mathbb{P}(\bar{A}) = 1- \mathbb{P}(A)$
  • $\mathbb{P}(A) = \mathbb{P}(A \cap B) + \mathbb{P}(A \cap \bar{B})$
  • $\mathbb{P}(A \cup B) = \mathbb{P}(A) + \mathbb{P}(B) - \mathbb{P}(A \cap B)$

Si $A\subset B$: * $\mathbb{P}(B\backslash A) = \mathbb{P}(B \cap \bar{A}) = \mathbb{P}(B) - \mathbb{P}(A)$ * $\mathbb{P}(B-A) = \mathbb{P}(B\bar{A}) = \mathbb{P}(B) - \mathbb{P}(A)$

Formule de Poincaré : $$\mathbb{P}\left(\bigcup_{i=1}^n A_i \right) = \sum_{i=1} \mathbb{P}(A_i) - \sum_{1\leq i<j \leq n} \mathbb{P}(A_i \cap A_j) + ... + (-1)^{n+1}\mathbb{P}\left(\bigcap_{i=1}^n A_i\right)$$

Permutations (3.1.1)

  • Il y a $n!$ permutations pour $n$ éléments distincts
  • "abc" donne abc, acb, bac, bca, cab, cba (6)
  • Pour $k$ éléments distincts, le $k^{\text{ème}}$ étant présent $n_k$ fois ($n=\sum n_k$), le nombre de permutations est : $$\frac{n!}{\prod_k n_k!}$$
  • "errer" donne eerrr, ererr, errer, errre, reerr, rerer, rerre, rreer, rrere, rrree ($10 = \frac{5!}{2!3!}$)

Arrangements (3.1.2)

  • On s'intéresse aux séquences de $p$ éléments pris parmi $n$. L'ordre d'apparition est important
  • Tirage successif de $p$ éléments sans remise: $$ A_n^p = \frac{n!}{(n-p)!}$$
  • Tirage successif de $p$ éléments avec remise : $$n^p$$

Combinaisons (3.1.3)

  • On s'intéresse aux sous-ensembles de $p$ éléments pris parmi $n$. L'ordre d'apparition n'est pas important
  • Tirage simultané de $p$ éléments parmi $n$: $$ C^p_n = \binom{n}{p} = \frac{n!}{p!(n-p)!}$$
  • Combinaisons de $p$ éléments parmi $n$ avec remise : $$ \frac{(n+p-1)!}{p!(n-1)!}$$
  • ensemble de trois symboles construits avec A et B : AAA, AAB, ABB, BBB ($4 = \frac{4!}{3!(2-1)!}$)

  • $\binom{n}{k} = \binom{n}{n-k$

Probabilité conditionnelle (1.6)

La probabilité de A sachant B est : $$ \mathbb{P}(A|B) = \frac{\mathbb{P}(A\cap B)}{\mathbb{P}(B)}$$

D'où : * $\mathbb{P}(A \cap B) = \mathbb{P}(A|B)\mathbb{P}(B) = \mathbb{P}(B|A)\mathbb{P}(A)$
* $\mathbb{P}(A|B) = \frac{\mathbb{P}(B|A)\mathbb{P}(A)}{\mathbb{P}(B)}$

Si $A\subset B$: * $\mathbb{P}(A|B) \geq \mathbb{P}(A)$ * $\mathbb{P}(B|A) = 1$

Théorème de Bayes (1.6.1)

Soit $(A_i){1\leq i\leq n}$ une partition de $\Omega$ $$ \mathbb{P}(B) = \sum^n \mathbb{P}(B \cap A_i) = \sum_{i=1}^n \mathbb{P}(B|A_i)\mathbb{P}(A_i)$$ A compléter

Indépendance statistique (1.7)

Deux évènements sont indépendants si la connaissance de l'uun ne donne aucune information sur l'autre : * $\mathbb{P}(A|B) = \mathbb{P}(A)$ * $\mathbb{P}(A\cap B)= \mathbb{P}(A)\mathbb{P}(B)$

Si A et B sont indépendants, $\bar{A}$ et $B$ le sont aussi

Attention : indépendant et exclusif ($A\cap B = \not 0$) n'ont rien à voir !
