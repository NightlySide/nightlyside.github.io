A propos

J'ai commencé la programmation dès l'âge de 9 ans avec un tout premier site web réalisé en flash. J'ai par la suite essayé de nombreux langages de programmation, tous avec un but différent pour répondre à ma curiosité. Je suis passé entre autres par le C/C++, Java, Python, HTML/CSS, PHP, Basic (sur calculatrice). Aujourd'hui j'utilise principalement Python pour ce qui relève de l'étude de données, création de courbres et du prototypage, mais dans mes heures creuses je code des pseudo-jeux en 2d en Haxe qui semble vraiment prometteur.
